# 🚗 Vehicle Resale Price Prediction System

## Live App
https://vehicle-resale-price-prediction.streamlit.app

## Description
Machine learning based web app to predict resale price of cars and bikes using Streamlit.

Users can:
- Select vehicle type (Car / Bike)
- Enter vehicle details
- Get predicted resale price
- View price trends using graphs

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Features
- Car price prediction
- Bike price prediction
- Interactive UI
- Graph visualization
- Real-time prediction
